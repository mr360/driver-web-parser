## driver-web-parser

### What is it?
Parsers driver websites for the latest drivers, also formats and organises the parsed driver data for easy importing into driverdb.

### List of driver parser compatible sites

- Intel
- AMD
- Nvidia
- Realtek
- Atheros
- Broadcom
- TP-Link
- Qualcomm
- Mediatek
- Texas Instruments
- Ralink
- Conexant
- IDT
- Synaptics
- Logitech
- Kionix
- VIA
